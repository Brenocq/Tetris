# Tetris
<img src="./img/tetris.gif" height="200">
Simple tetris made in OpenGL

## Installation

#### Ubuntu
```shell
sudo apt-get install libglew-dev
sudo apt-get install libglfw3-dev
sudo apt-get install libglm-dev
```

## Running

```shell
https://github.com/Brenocq/Tetris.git
cd Tetris
make
```
